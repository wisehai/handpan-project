常见问题

  • 应用没有声音怎么办？
    请检查设备音量、静音开关、蓝牙音频设备连接状态，并确认应用没有被系统限制音频播放。
  • 应用运行异常或闪退怎么办？
    请尝试重启应用或重启设备。如果问题仍然存在，请通过邮箱或 GitHub 提交反馈，并附上设备型号、iOS 版本和问题描述。
  • 如何反馈功能建议？
    你可以通过邮件或 GitHub Issues 提交建议。
